# Alura Challenge Front-End
Desafio Front-end da Alura. O desafio tem o objetivo de transformar o design em código web.

## Tecnologias e metodologias
- HTML;
- Vanilla CSS;
- Vanilla JS;
- Mobile First;
- Metodologia Kandan.
